Creating SQL Query Input in Atlas: Designing a Virtual Schema for Query Transformation

Published in Instituto Superior Técnico, University of Lisbon, 2025

This Master's thesis explores the integration of a new SQL query input mechanism for data retrieval within Atlas, an enterprise architecture platform. To address limitations of the previous querying system, this work introduces a Virtual Schema that enables intuitive SQL query formulation while preserving compatibility with the platform’s relational model — resulting in a 96% improvement in query execution performance.
